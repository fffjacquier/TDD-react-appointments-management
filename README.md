# appointments

Run npm install after getting the files

## Running tests

run all tests

```bash
npm test
```

Run server tests (express)

```bash
npm run test-server
```

## Building and running

Build

```bash
npm run build
```

Build server

```bash
npm run build-server
```

Then start the server

```bash
PORT=8000 npm run serve
```

Then test on http://localhost:8000/ on a browser
